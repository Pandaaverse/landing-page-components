# @pandaverse/landing-page-components

Components for Landing Page

## Installation

```bash
bun add @pandaverse/landing-page-components
```

## Usage

```tsx
import '@pandaverse/landing-page-components/styles.css';
import { Button } from '@pandaverse/landing-page-components';

function App() {
  return <Button>Click me</Button>;
}
```
